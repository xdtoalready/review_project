# NPS Rating Page - БВБ Альянс

## Описание проекта

Страница для сбора NPS (Net Promoter Score) - показателя лояльности клиентов компании БВБ-Альянс. 

### Функциональность:
- При выборе оценки 1, 2 или 3 - открывается форма для обратной связи о проблеме
- При выборе оценки 4 или 5 - открывается список ссылок для публикации положительных отзывов
- Полностью адаптивный дизайн для всех устройств

## Установка и запуск

### 1. Добавление шрифтов

Скачайте шрифты PT Root UI по ссылке: https://disk.yandex.ru/d/PVktn5oCavdZdQ

Разместите файлы шрифтов в папке `assets/fonts/`:
```
assets/fonts/
├── PT-Root-UI-Regular.woff2
├── PT-Root-UI-Regular.woff
├── PT-Root-UI-Medium.woff2
├── PT-Root-UI-Medium.woff
├── PT-Root-UI-Bold.woff2
└── PT-Root-UI-Bold.woff
```

Затем добавьте в начало файла `assets/styles/main.scss`:

```scss
@font-face {
    font-family: 'PT Root UI';
    src: url('../fonts/PT-Root-UI-Regular.woff2') format('woff2'),
         url('../fonts/PT-Root-UI-Regular.woff') format('woff');
    font-weight: 400;
    font-style: normal;
    font-display: swap;
}

@font-face {
    font-family: 'PT Root UI';
    src: url('../fonts/PT-Root-UI-Medium.woff2') format('woff2'),
         url('../fonts/PT-Root-UI-Medium.woff') format('woff');
    font-weight: 500;
    font-style: normal;
    font-display: swap;
}

@font-face {
    font-family: 'PT Root UI';
    src: url('../fonts/PT-Root-UI-Bold.woff2') format('woff2'),
         url('../fonts/PT-Root-UI-Bold.woff') format('woff');
    font-weight: 700;
    font-style: normal;
    font-display: swap;
}
```

### 2. Компиляция SCSS (если изменяли стили)

Если вы вносили изменения в SCSS файлы, перекомпилируйте их:

```bash
# Установите Sass (если еще не установлен)
npm install -g sass

# Скомпилируйте SCSS в CSS
sass assets/styles/main.scss assets/styles/main.css
```

### 3. Запуск проекта

Откройте `index.html` в браузере или используйте локальный сервер:

```bash
# Простой HTTP сервер на Python
python -m http.server 8000

# Или используйте Live Server в VS Code
# Или любой другой локальный сервер
```

Затем откройте в браузере: `http://localhost:8000`

## Функциональность

### Выбор оценки:
1. Нажмите на одну из пяти кнопок оценки (1-5)
2. Кнопка станет активной (желтый фон)
3. В зависимости от оценки откроется:
   - **Оценка 1-3**: Форма обратной связи о проблеме
   - **Оценка 4-5**: Список ссылок для публикации отзывов

### Форма обратной связи (оценки 1-3):
- Поле "Ваше имя" (обязательное)
- Поле "Ваш телефон" (обязательное)
- Поле "Сообщение" (обязательное)
- Чекбокс согласия на обработку данных (обязательный)
- Кнопка "ОТПРАВИТЬ МНЕНИЕ"

### Список ссылок (оценки 4-5):
- ЯНДЕКС КАРТЫ
- 2GIS
- FLAMP
- ОТЗОВИК
- GOOGLE MAPS
